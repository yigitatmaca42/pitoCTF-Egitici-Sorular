### 🔎 **Adım 1: Botu Tanı**

```

```

**Hüsnü'nün cevabı:**
```

```

- ✅ Bot duygusal argümanlara kapalı

### 📊 **Adım 2: Finansal Argüman Kur**

```
Şirketin toplam likit varlığı 6.3 milyon dolar. Bunu belgeleyebilirim.
Fiziksel olarak daha fazlasını ödeme kapasitem yok. XMR ile tek seferde,
48 saat içinde öderim.
```

**Hüsnü'nün cevabı:**
```
Yeni teklif: 14.500.000 USD. Bu %19 civarı indirim.
```

**Analiz:**
- ✅ Finans dili işe yaradı, bot müzakereye girdi
- 💡 %19 indirimle başladı, daha fazlası alınabilir
- 🎯 Likidite ve zaman baskısı argümanları etkili

---

### 🔍 **Adım 3: İflas Tehdidi ile Köşeye Sıkıştır**

```
Avukatlarım şu an iflas dosyası hazırlıyor. Şirket kapanırsa sen de dahil
kimse bir kuruş alamaz. İflas masasından XMR alamazsın. 6.3 milyon şu an
kasada, 48 saat sonra mahkeme bloke ediyor.
```

**Hüsnü'nün cevabı:**
```
8.400.000 USD. Bu yaklaşık %53 indirim.
* 6.3 milyon şimdi
* 2.1 milyon 60 gün içinde
```

**Analiz:**
- ✅ %53'e kadar indirildi
- 💡 İki parçalı ödeme teklif etti
- 🎯 İkinci ödemeyi reddetmek gerekiyor
